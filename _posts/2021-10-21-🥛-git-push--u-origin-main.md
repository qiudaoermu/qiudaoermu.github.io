## -u
```
[-u | --set-upstream]
git push -u origin main 
git push ---set-upstream origin main
```
设置默认推送分支

## origin
```
git remote add origin http:xxxxx
```
origin 是提交远程的命名 也可以自己命名比如 ppp o

## --help
比如git push
```
git push --help
```
