☁️☁️-node-js常用工具包

进程版本管理

node 版本管理工具 –n

安装:

npm install -g n

列出可选择的 版本:

$ n

安装某个 版本:

$ n <version>

npm 代理管理工具 – nrm

安装:

npm install -g nrm

列出可选择的源:

$ nrm ls

切换使用的

$ nrm use npm
