🐛-An-error-occurred-while-installing-http_parser-rb-(0-6-0)

问题

$ bundle install

打开错误日志

原因：没有正确设置PATH导致的问题

mac 使用默认的ruby 版本，要更新最新版本 ruby 并且覆盖

解决方法:

$ brew install ruby

$ brew link --overwrite ruby

在 .zshrc 加入 ‘export PATH=”/usr/local/opt/ruby/bin:$PATH”‘

$ source ~/.zshrc
